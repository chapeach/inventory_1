การอัพโหลดไฟล์ CSV

file csv ที่จะใช้จะประกอบไปด้วย 6 column
1. column A : คือ S/N หรือ path
2. column B : คือ ชื่อของอุปกรณ์นั่นๆ
3. column C : ระบุ location ที่นำอุปกรณ์ออก
4. column D : ระบุ location ที่นำอุปกรณ์เข้า
5. column D : จำนวนอุปกรณ์ที่นำออก
6. column F : จะ remark อะไรลงไปก็ได้

หลังจากเตรียม csv file เสร็จแล้ว ให้ไป upload file ที่หน้า (add csv file)
